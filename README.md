# Программа для тренировки сотрудников экстренных служб

3 модуля - теория, тест и практика

## Сборка

1. Запустить TrainingApp.pro в среде разработки QT Creator
2. Выбрать комплект (QT 6)
3. Сборка -> Собрать проект "TrainingApp"

Первый мой более-менее серъёзный проект, созданный до знания git и бэкапов, отчего много кода оказалось утеряно.
